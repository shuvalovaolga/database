DROP INDEX

Описание
Синтаксис
Примеры
Смотри так же

Описание

DROP INDEX удаляет определенный пользователем индекс из базы данных.

Индекс может быть удален его создателем и пользователем SYSDBA.

Важно: Индексы определенные системой, такие как для UNIQUE, PRIMARY KEY и FORGIN KEY ограничений целостности, не могут быть удалены.

Индекс в использовании не может быть удален, пока не освободится.

Синтаксис

DROP INDEX name;
Аргумент Описание
name; Имя существующего индекса.
