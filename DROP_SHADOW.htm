DROP SHADOW

Описание
Синтаксис
Примеры
Смотри так же

Описание

DROP SHADOW удаляет теневой набор и отсоединяет из теневого процесса. Команда SHOW DATABASE может быть использована, чтобы просмотреть элементы теневого набора базы данных.

Тень может быть удалена ее создателем и пользователем SYSDBA.

К началу страницы...

Синтаксис

DROP SHADOW set_num;

Обратите внимание:

Аргумент Описание
set_num Положительное целое число идентифицирующие существующий теневой набор.

Примеры

Следующая инструкция удаляет теневой набор базы данных:

DROP SHADOW 1;
