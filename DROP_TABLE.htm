DROP TABLE

Описание

DROP TABLE удаляет данные, метаданные и индексы таблицы из базы данных. Эта инструкция так же удаляет любые триггеры, которые обращаются к таблице.

Таблица, на которую ссылаются выражения SQL, в образы, ограничения целостности или сохраненные процедуры, не может быть удалена. Таблица, используемая в активных транзакциях, не может быть удалена, пока ее не освободят.

Обратите внимание: Когда удаляется внешняя таблица, DROP TABLE только удаляет определение таблицы из базы данных. Внешний файл не будет удален.

Таблица может быть удалена ее создателем и пользователем SYSDBA.

Синтаксис

DROP TABLE name;
Аргумент Описание
name Имя существующей таблицы.
