SET NAMES

Описание
Синтаксис
Примеры
Смотри так же

Описание

SET NAMES определяет кодировку используемую для последующих вложений базы данных. Это позволяет ISQL отменить кодировку базы данных по умолчанию.

SET NAMES должна применятся перед инструкцией CONNECT, чтобы она имела эффект.

К началу страницы...

Синтаксис

SET NAMES [charset];
Аргумент Описание
charset Имя кодировки, которая определяет текущую кодировку для данного процесса. По умолчанию: NONE.

Примеры

Следующая инструкция демонстрирует использование SET NAMES:

SET NAMES LATIN1;
CONNECT "employee.gdb";
