DROP VIEW

Описание
Синтаксис
Примеры
Смотри так же

Описание

DROP VIEW позволяет создателю вида удалить определение вида из базы данных если вид не используется в других видах, сохраненных процедурах или определении CHECK ограничения.

Вид может быть удален его создателем и пользователем SYSDBA.

К началу страницы...

Синтаксис

DROP VIEW name;

Обратите внимание:

Аргумент Описание
name Имя существующего определения вида, для удаления.

Примеры

Следующая инструкция удаляет определение вида:
