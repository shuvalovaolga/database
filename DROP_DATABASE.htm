Описание

Инструкция DROP DATABASE удаляет текущую присоединенную базу данных, включая все ассоциированные вторичные, теневые и log файлы. Удаление базы данных удаляет все данные, которые она содержит.

База данных может быть удалена ее создателем и пользователем SYSDBA.
