BEGIN DECLARE SECTION

Описание

BEGIN DECLARE SECTION используется во внедренных SQL приложениях, чтобы идентифицировать начало объявления переменных базового языка, для переменных, которые будут использованы в последующих инструкциях SQL. BEGIN DECLARE SECTION это также деректива препроцессора, которая указывает gpre, чтобы автоматически объявить переменную SQLCODE.

Важно: BEGIN DECLARE SECTION должна всегда появляться внутри секции объявления глобальных переменных модуля.

Эта инструкция может быть использована в SQL.

К началу страницы...

Синтаксис

BEGIN DECLARE SECTION;

Примеры

Следующие внедренные инструкции SQL объявляют секцию и переменную базового языка.

EXEC SQL
  BEGIN DECLARE SECTION;
    BASED ON EMPLOYEE.SALARY salary;
EXEC SQL
  END DECLARE SECTION;
