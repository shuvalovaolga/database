DROP FILTER

Описание
Синтаксис
Примеры
Смотри так же

Описание

DROP FILTER удаляет объявление BLOB фильтра из базы данных. Удаление декларации BLOB фильтра из базы данных не удаляет фильтр из связанной библиотеки, но делает фильтр недоступным из базы данных. Если определение (фильтра) удалено, любое приложение, которое использует фильтр, будет возвращать ошибку времени исполнения.

DROP FILTER завершается неудачей и возвращает ошибку, если любой процесс использует фильтр.

Фильтр может быть удален его создателем и пользователем SYSDBA.

Важно: BLOB фильтры не доступны для баз данных расположенных на серверах NetWare. Если BLOB фильтр случайно объявлен для базы данных на сервере NetWare, инструкция DROP FILTER должна быть использована, чтобы удалить декларацию.

Синтаксис

DROP FILTER name;
Аргумент Описание
name Имя существующего BLOB фильтра.
