Описание

PREPARE подготавливает инструкцю DSQL для повторного выполнения:

  • Проверяет инструкцию на синтаксические ошибки.
  • Проверяет типы данных факультативных динамических параметров.
  • Оптимизирует выполнение инструкции.
  • Компелирует инструкцию для выполнения с помощью EXECUTE.

PREPARE это одна из группы инструкций, которые подготавливают инструкции DSQL для выполнения.

Инструкция Назначение
PREPARE Подготавливает инструкцию DSQL для выполнения.
DESCRIBLE Заполняет XSQLDA информацией об инструкции.
EXECUTE Выполняет предварительно подготовленную инструкцию.
EXECUTE IMMEDIATE Подготавливает инструкцию DSQL, выполняет один раз, и сбрасывает ее.

После того, как инструкция подготовлена, она доступна для многократного выполнения, по мере необходимости в течении текущей сессии. Чтобы подготовить и выполнить инструкцию только однажды, используйте EXECUTE IMMEDIATE.

Инструкция PREPARE устанавливает символьное имя для подготавливаемой инструкции DSQL. Она не декларирует переменную базового языка. За исключением программ на языке C, gpre не различает между верхним и нижним регистром в инструкции, "B" и "b" обрабатываются как один и тот же символ. Для C программ, используйте gpre -either_case ключ, чтобы активизировать чуствительность к регистру в течении предвварительной обработки (preprocessing).

Если используется факультативное предложение INTO, PREPARE так же заполняет extended SQL descriptor area (XSQLDA) информацией о типе данных, длине и имени QWERTY...
