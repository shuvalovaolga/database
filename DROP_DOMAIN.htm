DROP DOMAIN

Описание
Синтаксис
Примеры
Смотри так же

Описание

DROP DOMAIN удаляет уже существующий домен из базы данных.

Если домен используется в любом описании столбца базы данных, операция DROP терпит неудачу. Для предотвращения сбоя (ошибочной ситуации), используйте ALTER TABLE для удаления столбцов базирующихся на домене, затем выполняйте DROP DOMAIN.

Домен может быть удален или его создателем или пользователем SYSDBA.

К началу страницы...

Синтаксис

DROP DOMAIN name;
Аргумент Описание
name Имя сушествующего домена.
